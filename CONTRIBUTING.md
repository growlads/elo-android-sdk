# Contributing

Thanks for your interest in the Elo Android SDK. This repository is the **distribution** for the SDK — the published artifact and a runnable quickstart sample. The SDK itself is developed in a separate (private) source repository, so the kinds of changes accepted here are narrow.

## Reporting bugs

Please open an issue using the **Bug report** template. Include the SDK version, Android API level, device, and a minimal repro.

For integration help, use the **Integration question** template instead.

## Security issues

Do **not** open a public issue. See [SECURITY.md](./SECURITY.md).

## Pull requests

PRs are welcome for the editable surface of this repo:

- `README.md` — consumer-facing integration docs.
- `samples/` — the runnable quickstart.
- `LICENSE`, `.gitignore`, `CHANGELOG.md`, this file, and `.github/`.

If you want to fix something inside the SDK itself, file a bug here and we'll patch it upstream and re-publish.

## Code of conduct

By participating, you agree to abide by the [Code of Conduct](./CODE_OF_CONDUCT.md).
